This crate explores genetic programming. Genetic programming is

a technique whereby computer programs are encoded as a set of genes that are then modified (evolved) using an evolutionary algorithm.

The problem we are trying to tackle with genetic programming is that of landing a moon lander. I.e. land a moon lander safely on the surface of a moon without user intervention. A nice interactive game that gives a feel for the problem can be found here.
